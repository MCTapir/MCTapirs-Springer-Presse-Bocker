## Installieren bei Chrome

1. Downloade die Erweiterung
2. Entpacke die Erweiterung
3. gehe auf chrome://extensions
4. Wenn noch nicht aktiviert ist, aktiviere den Developer Mode
5. Lade die entpackte Erweiterung
